# Project hero carousel

The primary image on every transformation detail page uses the project gallery order.

- Previous and next controls wrap around the gallery.
- Dots below the image allow direct selection.
- Horizontal swipe is available on touch devices.
- Left and right arrow keys work when the carousel has focus.
- Selecting the image opens the existing lightbox at the currently visible slide.
- Controls and dots are hidden when a project has only one image.
